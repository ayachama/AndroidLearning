# KotlinBasics

A standalone Kotlin project for learning and practicing core Kotlin language fundamentals. There is no Android Activity — the code runs as plain Kotlin programs with a `main()` entry point.

## Key Concepts

- **Kotlin entry point**: Using a top-level `fun main()` function as the program entry point
- **Console output**: Printing to standard output with `println()`
- **Variables**: Declaring immutable values with `val`
- **String templates**: Embedding expressions inside strings using `$variable` and `${expression}` syntax

## Main Files

| File | Purpose |
|------|---------|
| `HelloKotlin.kt` | Minimal "Hello Kotlin" program |
| `Practice.kt` | Sandbox for experimenting with Kotlin expressions and arithmetic |
