# MyApplication3

A simple Kotlin project used as a scratchpad for learning basic Kotlin syntax. There is no Android Activity — the program runs from a plain `main()` function.

## Key Concepts

- **Kotlin entry point**: Defining a top-level `fun main()` as the program start
- **Console output**: Using `println()` to print text to standard output
- **Basic project structure**: Understanding how a Kotlin-only module is set up inside an Android Studio project

## Main File

`HelloKotlin.kt` — Prints `"Hello, Kotlin!"` to the console when run.
