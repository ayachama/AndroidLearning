# MyApplication

A minimal Android starter project demonstrating the foundational structure of an Android application.

## Key Concepts

- **AppCompatActivity**: The base class for activities that provides backwards-compatible Material Design features
- **Activity lifecycle**: Overriding `onCreate()` as the entry point of an Activity
- **XML layouts**: Loading a UI defined in XML via `setContentView(R.layout.activity_main)`
- **Saved instance state**: Accepting the `savedInstanceState` Bundle parameter in `onCreate()`

## Main File

`MainActivity.kt` — A bare-bones Activity that inflates `activity_main.xml` as its content view.
